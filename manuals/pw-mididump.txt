.TH "PW-MIDIDUMP" 1 "" "" "General Commands Manual"
.SH NAME
pw-mididump \- The PipeWire MIDI dump
.SH SYNOPSIS
.nf
\fBpw\-mididump\fP [\fIoptions\fP] [\fIFILE\fP]
.fi
.sp
.SH DESCRIPTION
.sp
Dump MIDI messages to stdout.
.sp
When a MIDI file is given, the events inside the file are printed.
.sp
When no file is given, \fBpw\-mididump\fP creates a PipeWire
MIDI input stream and will print all MIDI events received on the port to
stdout.
.SH OPTIONS
.INDENT 0.0
.TP
.B \-r | \-\-remote=NAME
The name the remote instance to monitor. If left unspecified,
a connection is made to the default PipeWire instance.
.TP
.B \-h | \-\-help
Show help.
