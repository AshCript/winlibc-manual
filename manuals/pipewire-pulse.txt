.TH "PIPEWIRE-PULSE" 1 "" "" "General Commands Manual"
.SH NAME
pipewire-pulse \- The PipeWire PulseAudio replacement
.SH SYNOPSIS
.nf
\fBpipewire\-pulse\fP [\fIoptions\fP]
.fi
.sp
.SH DESCRIPTION
.sp
\fBpipewire\-pulse\fP starts a PulseAudio\-compatible daemon that integrates with
the PipeWire media server. This daemon is a drop\-in replacement for the
PulseAudio daemon.
