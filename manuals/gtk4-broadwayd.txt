.SH NAME
gtk4-broadwayd \- The Broadway display server
.SH SYNOPSIS
.nf
\fBgtk4\-broadwayd\fP [OPTIONS...] <DISPLAY>
\fBgtk4\-broadwayd\fP \-\-port=PORT \-\-address=ADDRESS <DISPLAY>
\fBgtk4\-broadwayd\fP \-\-unixaddress=ADDRESS <DISPLAY>
.fi
.sp
.SH DESCRIPTION
.sp
\fBgtk4\-broadwayd\fP is a display server for the Broadway GDK backend. It allows
multiple GTK applications to display their windows in the same web browser, by
connecting to gtk4\-broadwayd.
.sp
When using gtk4\-broadwayd, specify the display number to use, prefixed with a
colon, similar to X. The default display number is 0.
.INDENT 0.0
.INDENT 3.5
.sp
.nf
.ft C
gtk4\-broadwayd :5
.ft P
.fi
.UNINDENT
.UNINDENT
.sp
Then point your web browser at \fBhttp://127.0.0.1:8085\fP\&.
.sp
Start your applications like this:
.INDENT 0.0
.INDENT 3.5
.sp
.nf
.ft C
GDK_BACKEND=broadway BROADWAY_DISPLAY=:5 gtk4\-demo
.ft P
.fi
.UNINDENT
.UNINDENT
